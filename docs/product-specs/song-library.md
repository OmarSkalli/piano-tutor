# Song Library

A browsable list of songs available to the student, displayed alphabetically by title. Each song is clickable and opens a Song View for that song.

## Problem

Students need a way to discover and access the songs available in the app. Without a library view, there's no entry point to begin practicing.

## Solution

A Song Library screen that lists all available songs alphabetically by title. Tapping/clicking a song navigates to its Song View.

## Requirements

- Display all songs from the library as a list, sorted alphabetically by title
- Each list item shows the song title
- Clicking a song opens the Song View for that song
- List updates automatically when new songs are added to the library

## Success Criteria

- All songs in `library.json` appear in the list
- Songs are sorted A–Z by title
- Clicking a song opens the correct Song View

## Out of Scope

- Song search or filtering
- Song metadata display (composer, difficulty, etc.) — may be added later
- Song View implementation (separate spec)
