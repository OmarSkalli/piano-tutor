# Piano Tutor

A Vite app for learning and practicing piano — import MIDI songs, study them visually, and practice with a connected MIDI keyboard and real-time feedback.

## Quick Start

Piano Tutor has two modes: **Study** (visualize a song on a keyboard, step through notes, hear playback) and **Practice** (connect a MIDI keyboard and play along with real-time correct/wrong feedback). Import any MIDI file to get started.
